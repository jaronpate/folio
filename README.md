# Folio Nuxt

This is a Nuxt.js port of Jaron's portfolio.

## Development

Make sure you have [Bun](https://bun.sh) installed.

```bash
# Install dependencies
bun install

# Start development server
bun run dev
```

## Production

```bash
# Build the project
bun run build

# Preview the production build
bun run preview
```
